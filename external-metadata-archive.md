# CPAN Meta v3 External Metadata Archive

There may be a desire to support a separate archive file for distributions.

A use case for this is to generate metadata for legacy CPAN distributions.

The format of these archives, the filenames, and how and where they would be hosted to be determined.

Presumably the name would be similar to a CPAN distribution, and the content would be files and directories that would normally be in the distribution.
