# Metadata c3 Well-Known Names

The following filenames in the [metadata directory](metadata-directory.md) are reserved:

- [automation-policy.json](automation-policy.md)

In cases where a specific metadata specification requires files to be in a different location,
tools for META v3 would know to lokk in alternative locations.

The following names and directories in the distribution and are software repository root reserved for known external tools:

- docs

  Used for community health documentation that does not reside in the document root.

- LICENCES

  Used by the REUSE specification.
